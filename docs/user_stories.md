# User Stories

## Account Creation & Verification
**As a** UTRGV student
**I want** to create an account using my UTRGV email and phone number
**So that** I can verify my identity as a real student and access the app securely

## Personalized Profile & Interests
**As a** user
**I want** to customize and create my profile with my interests and hobbies
**So that** I can network with others who are like me

## Facial Recognition Login
**As a** returning user
**I want** to login using facial recognition
**So that** I can access my account quickly and securely without having to type in anything

## Posting and Engagement
**As a** user
**I want** to create posts and interact with others posts
**So that** I can keep in touch with my friends and engage in the community at UTRGV

## Discover & Connect
**As a** student
**I want** to search for and follow other users based on their interested or name
**So that** I can grow my social circle on campus

## Firebase & Database Setup
**As a** developer
**I want** to configure Firebase and link it with the app’s database
**So that** user data is stored securely in the cloud

## UI Setup & Design
**As a** user
**I want** to have an easy to navigate interface
**So that** I can move between pages with ease just like in any other app

## Account Creation
**As a** UTRGV student
**I want** to sign up using my UTRGV email
**So that** only students that verified their account can use the platform

## Email & Phone Verification
**As a** new user
**I want** to verify my email and number through codes that are sent through sms
**So that** my account is secure

## Auto Login
**As a** frequent user
**I want** to stay logged in automatically
**So that** I don't have to enter my credentials every time I open the app

## Homepage Feed
**As a** user
**I want** to see posts from my friends
**So that** I can keep being updated about their activity

## Profile Creation & Editing
**As a** user
**I want** to create and customize my profile the way I want
**So that** others can know more about me

## Search (People, Interests, Hobbies)
**As a** user
**I want** to search for other classmates and students by name or interests
**So that** I can find people with similar hobbies as mine

## Follow/Unfollow Users
**As a** user
**I want** to follow and unfollow other users
**So that** I can manage who I see on my feed

## Friend Suggestions
**As a** user
**I want** to receive friend suggestions
**So that** I can find people more easily based on mutual friends or similar hobbies

## Create a Post
**As a** user
**I want** to create a post with photos, videos, and text
**So that** I can share my best moments with others

## Like/Comment/Share Posts
**As a** user
**I want** to share my posts, have people like them, and comment on them
**So that** we all can engage with each other's content

## Notifications
**As a** user
**I want** to receive push notifications when someone interacts with anything on my profile 
**So that** I can stay updated on the app without having to constantly go on it

## Temporary Posts (Stories)
**As a** user
**I want** to post temporary content that doesn't appear anymore after 24 hours
**So that** I can just share the moments briefly without them being permanently there on my profile

## Close Friends Posts
**As a** user
**I want** to share posts only with my close friends
**So that** I can control who sees my personal updates

## Explore Page
**As a** user
**I want** to explore trending or public posts
**So that** I can discover new people and content

## “For You” Recommendations
**As a** user
**I want** to see recommended posts based on my interests and interactions
**So that** I can discover content that fits me personally

## Security & Data Integrity
**As a** system administrator
**I want** to ensure all data is encrypted and user privacy is protected
**So that** the app complies with data protection standards
